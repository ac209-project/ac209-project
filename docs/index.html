Introduction

The purpose of this project is to use publicly available data through Spotify's API, the Spotipy python library, and audio features in the Million Song Dataset (MSD) to predict the success of Spotify user playlists, and create a tool to build successful playlists based on user specified features (genre, artist, etc.).

Throughout this project "playlist success" (our response variable) is defined as the number of playlist followers CORRECT ME IF I'M WRONG, BUT I THINK THAT'S WHERE WE LANDED. We have employed machine-learning regression techniques taught throughout the semester as well as Word2vec modeling to predict the success of BLANK NUMBER OF playlists.

Data Collection

Baseline data was collected primarily from two sources - the Spotify Web API and the MSD. After sourcing the Spotify users who “own” (the playlists are attached to their accounts) a large number of playlists, the spotipy library was employed to collect user playlists, tracks, user followers, and genre data from the API, each saved as JSON files. Genre information for each playlist was harder to source and incorporate than one would think. It does not come directly from each playlist, but rather from the artists of each track as a separate query. Before combining, the separate files have the following basic form:

Data File Observations Features
Usernames 8466 1
User Followers 8457 1
Playlists 8231 7
Genres 39998 5
Tracks 367949 8

The files were combined and manipulated into a single DataFrame via the playlist ID, username, and artist name attributes. While cleaning, we decided to eliminate any playlist with only one follower (the owner).

Exploratory Data Analysis

First look at playlist success:
Engineering more features around artist and playlist data yielded interesting relationships. A cursory measure of playlist success, our generalized project goal, is the number of followers per playlist. The histograms to the right show a significant right-skew, with most playlists having few followers and a few approaching the 1M mark. The log-transform of followers illustrates a potentially more useful response variable (see plots to the right).

Categorizing artists:
We further aggregated the artists by splitting them up into five distinct categories using combinations of their mean popularity (as determined by Spotify's popularity metric) and total number of artist tracks as follows:
