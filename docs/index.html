Introduction

The purpose of this project is to use publicly available data through Spotify's API, the Spotipy python library, and audio features in the Million Song Dataset (MSD) to predict the success of user playlists, and create a tool to build successful playlists based on user specified features (genre, artist, etc.).

Throughout this project playlist "success" is defined as FILL IN HERE. We have employed machine-learning regression techniques taught throughout the semester as well as Word2vec modeling to predict the success of BLANK NUMBER OF playlists.
